Projets


Lors de mes années de BTS, la réalisation de plusieurs projets m'a permis de développer mes compétences techniques et de me familiariser avec les outils professionnels. Je vais vous en présenter quelques-uns.

Projet Aunis Freeware :

Ce projet devait être réalisé en équipe de deux sur Visual Studio Code en HTML/CSS et en PHP. L'objectif était de créer un site web pour une association fictive nommée Aunis Freeware, avec un cahier des charges précis. Le site devait inclure des fonctionnalités telles qu'une page d'accueil, une page de présentation de l'association, une galerie d'images, une section pour les actualités, et un formulaire de contact.

⮕Lien vers le GitHub du projet

Projets Cisco Packet Tracer :

Lors de différents projets avec Cisco Packet Tracer, j'ai pu apprendre à configurer des réseaux informatiques de différentes tailles et complexités. J'ai aussi appris à gérer les bases de la gestion de réseau, comme le routage, la commutation, la sécurité réseau et la gestion des adresses IP.

Projet GLPI :

Ce projet devait être réalisé seul avec GLPI. L'objectif de ce projet était de mettre en place une gestion des incidents avec GLPI pour une entreprise fictive.

⮕Lien vers le compte rendu du projet

Projet RPG :

Ce projet devait être réalisé en équipe de trois sur éclipse en java, ainsi j'ai réalisé ce projet avec deux autres camarades. Nous devions produire un RPG. L'objectif était de créer un RPG où les joueurs peuvent évoluer dans un monde, accomplir des quêtes, gérer des objets et faire face à des événements aléatoires. Le jeu doit être interactif et permettre aux joueurs de prendre des décisions qui influencent leur progression.

⮕Lien vers le GitLab du projet

⮕Lien vers le compte rendu du projet

Projet CDM :

Ce projet devait être réalisé seul avec Symfony. Ce projet avait pour objectif de créer une application web de gestion d'une coupe du monde de football. L'application devait permettre aux utilisateurs de s'inscrire, de consulter les équipes participantes, les matchs et leurs scores. Les administrateurs pouvaient ajouter, modifier et supprimer des équipes et des matchs via une API Api Platform.

⮕Lien vers le GitLab du projet

Projet TheMovie.db :

Ce projet devait être réalisé en équipe de quatre sur Android Studio en Java. Nous devions réaliser une application mobile qui permet aux utilisateurs de rechercher des films et des séries à partir d'une base de données externe (TheMovieDB). L'application devait permettre d'afficher des détails supplémentaires sur chaque film ou série renseignée sur l'API.

⮕Lien vers le GitLab du projet

Projets Gestion de BDD :

Lors de différents projets, j'ai eu de la gestion de base de données avec MySQL, j'ai pu apprendre à concevoir, créer et gérer des bases de données relationnelles. J'ai travaillé sur la modélisation des données, la création de schémas, l'écriture de requêtes SQL pour manipuler les données, et aussi la création de schémas de base de données (UML).
